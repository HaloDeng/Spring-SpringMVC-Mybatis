1.环境：
jdk 1.8
maven 3.3.9
tomcat 7.0
数据库 mysql
所有文件编码格式为utf-8
框架版本在pom文件中有
注意：spring的包最好一致，不然可能会出现很多莫名其妙的问题
接口访问路径例如：
http://localhost:8080/MyBatisMVC/user/userInfo.do
